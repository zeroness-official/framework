## overview

zeroness は、複雑な時代における「シンプルな原則」を探求するプロジェクトです。

- **zero** は、余白・空（くう）・手放し。
- **one** は、輪郭・始まり・自分。
- **zone** は、流れ・世界・共鳴。

それぞれは、独立した概念ではなく、常に互いを必要とする存在。
三位一体のように巡りながら、私たちの在り方を静かに照らします。
